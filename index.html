IPL Top 4 Predictor

Predicting IPL Playoff Teams Using Machine Learning

About This Project

IPL Top 4 Predictor is a data-driven web application developed as part of the Machine Learning and Deep Learning Premier League (MDPL), a national-level datathon organized by Thiagarajar School of Management (TSM), Tamil Nadu.

The project uses a Machine Learning model trained on historical IPL data to predict the Top 4 playoff teams and eliminated teams of the current season. The model is deployed on a Python-based FastAPI backend and communicates with the frontend via a REST API.

Users can manually input team performance data (Matches, Wins, Losses, Points, NRR, Remaining Matches), which is converted into a CSV file and submitted to the backend. The predictions returned by the ML model are then displayed interactively through a clean and responsive React-based frontend.

This project highlights real-world applications of Machine Learning, sports analytics, and web deployment, and serves as a showcase of our team's technical skills in solving competitive data challenges.

Usage Guide

  1. Enter current performance data for each IPL team in the input table.
  2. Click on “Generate & Submit CSV” to send the data to the backend model.
  3. Please wait for 30–40 seconds while the model processes the input and generates predictions.
  4. The system will return and display the predicted Top 4 and eliminated teams.
  5. Click “Refresh” to rerun predictions with updated data.
  6. Use the “Reset Form” button to clear all inputs and start over.
